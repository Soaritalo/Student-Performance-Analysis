# Student-Performance-Analysis
This project analyzes students' grades and generates performance statistics such as:  Individual average  Class average  Ranking (highest to lowest performance)  The goal of this project is to practice data structures, loops, sorting, and basic statistical calculations using Python.

Script Python que analisa notas de alunos e gera estatísticas de desempenho.

## O que faz

- Calcula a média individual de cada aluno
- Calcula a média geral da turma
- Gera um ranking de desempenho (do maior para o menor)

## Como usar

```bash
python main.py
